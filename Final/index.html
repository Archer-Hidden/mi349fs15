Docker: what is DeviceMapper

DeviceMapper since Linux 2.6 was introduced in Linux has become the most important technology. It supports common device mapping mechanism Logical Volume Manager in the kernel, which is used for the realization of storage resource management block device driver provides a highly modular kernel architecture, it contains three important objects concept, Mapped Device, Mapping Table , Target device.
