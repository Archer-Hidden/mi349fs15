Clean TD game frame

    No extra package used, pure javascript+html5 , basic game frame for you to start a prefect TD gamme!

    Since it's just a frame, so no image/anime is load, you can only see a world with dots,bars and rectangles. But no worries, you can always easily make it fancy by add your images. The code is easy to read! :)

    The black dots on the screen are the enemies, which will follow the green route, heading to the red square in the center, which is your base.

    Interface preview:

    Each tower(black rectangle) will shoot a bullet (blue dot) at certain frequency. Each bullet has only one target, and will not change target. All the died data (EX dead enemies and dead bullet) will be cleaned out quickly.
